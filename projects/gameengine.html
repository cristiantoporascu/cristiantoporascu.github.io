This project was developed for a coursework that was undergone during the 3rd of the Games Technology course at Coventry University, and it achieved a score of 91.4.

This game engine which I called FunEngine, developed using various APIs, that assited during the development of some subsystems or complete frameworks for subsystems like Audio System, Physiscs System and Networking System.

It was mainly built with compartmentalisation in mind using C++ and OOP. Makes use of various programming paterns like singletone, observer and state patterns. The Networking pemits the creation of a host and the connection of multiple clients, which transfer data using a dedicated host as the mean of transfer.

The UI was built so that it supports multiple inputs from the user that allows the creation of various levels and the exporting and importing as well as parsing of .XML files into game data and finally level data.
