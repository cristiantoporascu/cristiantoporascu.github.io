VITSOE PLANNING TOOL

  • Watch video

As it is a project made during my role as a Junior Web Developer for Vitsoe, the source code can't be provided.

This tool was developed as a way to improve the workflow of planning various configurations subsystems for the customers of Vitsoe Ltd. It's a web application built using React and is mainly aimed towards the use of customers.

My contribution on this project was the development of the functionality that was taking care of the interactions of the actual configurator. Which includes the dragging and the droping of the components on the set drawing, as well as the visual interaction with the tools, which describe the actions without having an actual description.

The project was build in 4 weeks time with a small focused team of 4 people. My main task was preparing the data set for the components and creating the basic (unstyled) markup for all pages aside from creating the interactions with the configurator.
